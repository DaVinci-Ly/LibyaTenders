# 🏛️ متتبع العطاءات الحكومية الليبية (Libya Tenders Tracker)

نظام محلي متكامل وبسيط لمتابعة وجلب العطاءات الحكومية الرسمية من منصة العطاءات التابعة لحكومة الوحدة الوطنية الليبية، مع تصنيفها ذكياً وتتبع الكلمات المفتاحية التي تهمك لتسهيل العثور على الفرص المناسبة.

---

## ✨ المميزات الرئيسية
* **🔄 جلب تلقائي فوري:** جلب أحدث العطاءات مباشرة من البوابة الرسمية (`attaat.pm.gov.ly`) عبر الـ API الرسمي.
* **🧠 تصنيف ذكي وتلقائي:** يقوم النظام بتصنيف العطاءات تلقائياً إلى مجالات متعددة (تقنية، إنشاء وبناء، كهرباء وطاقة، طبي وصحة، إلخ).
* **🔍 متتبع الكلمات المفتاحية:** إمكانية إضافة كلمات مفتاحية مخصصة (مثل: شبكات، برمجيات، حفر آبار) للحصول على تنبيهات وتحديد العطاءات المطابقة لاهتماماتك باللون الأخضر.
* **💻 تشغيل بضغطة زر واحدة:** ملف تشغيل ذكي ومخصص لنظام الماك يقوم بفحص المنفذ، إغلاق النسخ السابقة، وتشغيل الخادم تلقائياً.
* **🌐 متوافق مع متصفح Google Chrome:** يفتح النظام واجهته تلقائياً على متصفح جوجل كروم لتوفير أفضل تجربة تصفح.
* **🎨 واجهة مستخدم حديثة وممتازة:** تصميم داكن مريح للعين (Dark Mode) ذو طابع احترافي مع إحصائيات فورية وتصفية متقدمة.

## 🚀 طريقة التشغيل والاستخدام

### المتطلبات الأساسية:
* نظام تشغيل macOS.
* تثبيت لغة Python 3 (مثبتة افتراضياً على الماك).

### خطوات التشغيل:
1. اذهب لمجلد المشروع.
2. اضغط مرتين (Double-click) على ملف **`run_tracker.command`**.
3. ستفتح نافذة Terminal منسقة، وستقوم بتشغيل النظام وفتح الواجهة تلقائياً في متصفح **Google Chrome** على الرابط المحلي:
   `http://127.0.0.1:7845`
4. لإيقاف التشغيل، ببساطة أغلق نافذة الـ Terminal أو اضغط على `Ctrl + C` داخلها.

## 🛠️ التقنيات المستخدمة
* **الخلفية (Backend):** Python 3 (استخدام المكتبات القياسية `http.server` و `urllib` بدون أي إضافات خارجية لتسهيل النقل).
* **الواجهة (Frontend):** HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS) بدون أطر عمل لسرعة الأداء القصوى وخفة الحجم.
* **تصميم الألوان والخطوط:** خط Cairo من Google Fonts، وألوان غنية متناغمة مع المظهر الداكن العصري.
